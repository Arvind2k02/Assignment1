1. <strong>
    • Used to indicate strong importance or emphasis on text.
    • Best practice is to use it for important text that should be visually emphasized, such as headings or keywords.
  2. <em>
    • Used to indicate emphasis on text.
    • Best practice is to use it for text that should be emphasized or highlighted, such as emphasizing a point in a paragraph.
  3. <u>
    • Historically used to underline text.
    • Best practice is to avoid using it for underlining text as it can be confused with hyperlinks. Instead, use CSS to style text with underlines if necessary.
  4. <del>
    • Used to indicate deleted or removed text.
    • Best practice is to use it when showing revisions or changes in documents, such as in version control systems.
  5. <ins>
    • Used to indicate inserted or added text.
    • Best practice is to use it when showing revisions or changes in documents, such as in version control systems.
