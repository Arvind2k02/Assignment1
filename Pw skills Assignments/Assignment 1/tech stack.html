1. HTML

    Primary Use Cases Key Features/Benefits
    • Building the structure of web pages
    • Simple and easy to learn
    • Compatible with all web browsers
    • Allows for semantic markup

  2. CSS

    Primary Use Cases Key Features/Benefits
    • Styling the appearance of web pages
    • Allows for separation of content and presentation
    • Supports various style properties and selectors
    • Enables responsive design and layout

  3. JavaScript

    Primary Use Cases Key Features/Benefits
    • Adding interactivity and dynamic behavior to web pages
    • Client-side scripting
    • Supports event handling and DOM manipulation
    • Allows for asynchronous programming
    • Rich ecosystem with libraries and frameworks
